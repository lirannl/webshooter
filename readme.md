# The Marvelous Webshooter
The goal of Webshooter is to enable remote access to machines using a web browser.

## Targets (in order):
- [ ] Simple authentication (asymmetric negtiation of a symmetric key, switch to a symmetric key)
- [ ] Hosting on wayland
- [ ] Front-end
- [ ] Hardware encoding
- [ ] Standard input handling (keyboard and mouse)
- [ ] Advanced input handling (gamepads, touchscreens (as touchscreens, rather than just emulating a mouse))
- [ ] Advanced capture handling (handling multi-monitor systems)
- [ ] Hosting on X11, Windows
- [ ] Hosting on MacOS, Android(?), iOS(???)
